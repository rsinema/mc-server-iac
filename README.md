# Project to Learn Terraform and AWS
This is a project that I am doing to learn Terraform and IaC, specifically with AWS since that is what I am most familiar with and I want to learn it better. 
This will have the Terraform code to build a EC2 instance to work as a Minecraft server, security groups for the server, lambda functions to start/stop 
the server to save on compute when not in use, lambda functions to allow people access to the server, a volume to hold the server data, possibly a elastic IP to associate 
with the server, etc.
